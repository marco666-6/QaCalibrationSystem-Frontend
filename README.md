# QA Calibration System Frontend

Frontend for the QA Calibration backend in `QaCalibrationSystem-Backend`.

## Available modules

- Authentication and password recovery
- Calibration dashboard
- Equipment management
- Master data management for sections, positions, and default locations
- User administration
- My profile and forced password change flow

## Development

```bash
npm install
npm run dev
```

The development API base URL is configured in `.env.development` and points to `http://localhost:5033/api`.
